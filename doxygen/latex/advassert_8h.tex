Advanced version of standard assert, capable of printing variables\textquotesingle{} values and names. 

If passed condition is true, nothing happens. Otherwise internal function \doxylink{advassert_8h_a7837447211dfe34718ffce5e7d4813df}{advassert\+\_\+()} is called, which prints passed variables\textquotesingle{} names and values and calls standard abort(). If NEDBUG is defined, all asserts are turned off. 
\begin{DoxyParams}{Parameters}
{\em cond} & Condition to check. \\
\hline
{\em expl} & Constant string message, explaining what this assert means. \\
\hline
{\em n\+Vars} & Number of variables being passed. \\
\hline
{\em ...} & Variables, which names and values must be printed, written as follows\+: (type) var\+\_\+name \\
\hline
\end{DoxyParams}
\begin{DoxyNote}{Note}
USAGE EXAMPLE 1\+: ADV\+\_\+\+ASSERT(i1 \texorpdfstring{$>$}{>} 0, \"{}i1 must be positive\"{}, 1, (int) i1); i1 has type int. 

USAGE EXAMPLE 2\+: ADV\+\_\+\+ASSERT(p != NULL, \"{}null pointer p\"{}, 2, (int \texorpdfstring{$\ast$}{*}) p, (int) some\+\_\+int); 

USAGE EXAMPLE 3\+: ADV\+\_\+\+ASSERT(0, \"{}\+Unreachable line\"{}, 0, \"{}\"{}); If you don\textquotesingle{}t want to pass any variables, pass 0 as number of variables and anything (for example empty string) instead of variables. 
\end{DoxyNote}
