Internal function implementing advanced version of standard assert().

MUSTN'T be used on its own, only as a part of the macro ADV_ASSERT(). Please, read docs concerning this macro to learn how to properly use this macro.

Parameters
[in]conditionAssertion condition converted to string by macro.
[in]explanationWritten by user explanation of the assertion condition as a constant string.
[in]fileFile where assertion failed, passed in by macro FILE.
[in]lineLine where assertion failed, passed in by macro LINE.
[in]vars_as_stringVariables to print passed by user, converted by macro to string.
[in]nVarsNumber of variables to print passed by user.
